ServerOnly

<ServerOnly>

Component that renders its children only if used on a Node.js server.

Usage

import {ServerOnly} from 'libreact/lib/ServerOnly';

<ServerOnly>
  <div>
    You can see this only on server.
  </div>
</ServerOnly>
